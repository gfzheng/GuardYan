# 守檐·GuardYan — 初赛作品说明

## 项目简介

守檐·GuardYan 是一枚硬币大小的老人随身应急呼救设备，通过 ESP32-C3 + MPU6050 实现跌倒检测与一键 SOS，配套 Web Dashboard 供子女/护工管理。核心设计理念：**不是让老人学 App，而是让技术缩小为一个红色按钮。**

## 技术架构

```
┌──────────────┐      Wi-Fi/MQTT       ┌──────────────────┐
│  GuardYan    │ ◄──────────────────► │  Web Dashboard   │
│  随身硬件    │                      │  (本作品在线体验) │
└──────────────┘                      └──────────────────┘
   ESP32-C3                              Vanilla JS
   MPU6050                               localStorage
   蜂鸣器+LED                            Chart.js
```

## 在线体验

👉 **Dashboard 在线体验链接**：`https://<你的用户名>.github.io/guardyan-dashboard/`

评审可直接点击体验：事件总览、老人档案、用药计划三页完整功能，右下角"模拟控制台"可触发跌倒/SOS/用药事件。

## 硬件运行说明

### 所需硬件

| 组件 | 型号 | 数量 |
|------|------|------|
| 主控 | ESP32-C3 开发板 | 1 |
| 运动传感器 | MPU6050 模块 | 1 |
| 按钮 | 12mm 红色轻触开关 | 1 |
| 蜂鸣器 | 有源蜂鸣器 | 1 |
| LED | 3mm 红色 LED | 1 |
| 麦克风 | 驻极体麦克风模块 | 1 |

### 烧录步骤

1. 安装 [Thonny IDE](https://thonny.org/) 并配置 MicroPython 解释器
2. 将 `guard-yan-firmware/` 目录中的文件烧录到 ESP32-C3
3. 修改 `config.py` 中的 `WIFI_SSID` 和 `WIFI_PASSWORD`
4. 运行 `main.py`

## 技术亮点

- **零学习成本设计**：单一大按钮，无屏幕，老人无需学习任何 App
- **BOM < 60 元**：量产成本可控，适合社区批量配发
- **MicroPython 快速开发**：Python 语法，开发周期短，代码可读性强
- **离线模拟模式**：Dashboard 无需硬件/网络即可完整体验，内置模拟控制台
- **五级状态机跌倒检测**：失重→撞击→姿态剧变→静置确认，含误报抑制

## 项目结构

```
guard-yan-dashboard/    # Web Dashboard（初赛评审主体验入口）
├── index.html          # 事件总览页
├── profile.html        # 老人档案页
├── medication.html     # 用药计划页
├── css/main.css        # 统一样式
└── js/                 # 数据层 + 组件 + 模拟器

guard-yan-firmware/     # ESP32 固件（MicroPython）
├── main.py             # 主循环
├── fall_detector.py    # 五级状态机跌倒检测
├── sos_handler.py      # SOS 按钮中断
├── mpu6050.py          # 传感器驱动
├── mqtt_client.py      # Wi-Fi + MQTT
└── ...
```
